# Deliberation Closure v0.1 Conformance Scenario

This oracle tests the [Deliberation Closure Profile](../../../spec/profiles/deliberation-closure/v0.1/README.md) with 23 semantic cases and 20 forbidden inferences. It treats closure as a receipt over independently effective adoption, rejection, tolerance, deferral, prompting-policy revision, target correction, and Intent revision artifacts.

Run `node conformance/scenarios/deliberation-closure-v0.1/validate.mjs`.

Passing does not prove participant understanding, fair governance, effect atomicity, or action authorization.
