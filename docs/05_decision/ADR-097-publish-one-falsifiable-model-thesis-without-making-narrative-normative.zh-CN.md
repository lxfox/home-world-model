# ADR-097：发布一份可证伪模型论纲，但不让叙事成为规范

- 状态：接受为公开解释入口；不改变规范 wire
- 日期：2026-07-19
- 英文规范：[`ADR-097-publish-one-falsifiable-model-thesis-without-making-narrative-normative.md`](ADR-097-publish-one-falsifiable-model-thesis-without-making-narrative-normative.md)

## 背景

可执行模型已经有正常形，但公众仍需穿过很长的 README、数十个 Profile 和许多 ADR。新共识需要一项紧凑主张，把家庭连续性、Agent 可读知识、三维表示、人／宠物／时间、装修和行动问责连接起来。若不从属于精确规范，叙事压缩也可能成为危险的隐藏 specification。

## 决定

1. 发布一份双语 Model Thesis，作为主要解释入口。
2. 把核心主张表达为由家庭控制、独立于某个 Agent、平台或设备图的可移植知识历史。
3. 用家庭语言解释概念核、3个行为契约、package binding、7条公理、8个扩展槽位和 observe–resolve–act–feedback 闭环。
4. 连接三维表示、installed influence、人／宠物／时间、anticipation、装修与采购，但不把它们晋升为 Core truth 或 authority。
5. 包含显式证伪与证据限制章节。外部异议保持为证据，不变成分数。
6. 论纲没有规范权威。精确 Core、Profile 与 Conformance Set 永远优先；冲突时修正文稿，不能用修辞解释 Schema。
7. 验证双语模型边界与 normative firewall。

## 后果

- 新读者可以先评估项目赌注，再阅读实现细节。
- 公开叙事从已测试正常形推导，不再只是累积 slogan。
- 后续传播可以引用一份稳定论纲，同时把每项规范主张链接到精确事实源。
- 叙事流行度不能改变 wire 语义、采用状态或共识证据。
