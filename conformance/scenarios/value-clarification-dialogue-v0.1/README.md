# Value Clarification Dialogue v0.1 oracle

This oracle tests the [Value Clarification Dialogue Profile](../../../spec/profiles/value-clarification-dialogue/v0.1/README.md) with 36 semantic cases and 40 forbidden inferences. It covers decision influence, burden, interruption, privacy, neutrality, exact response binding, opt-out meanings, scope, authority, correction, and bounded value inference.

Run `node conformance/scenarios/value-clarification-dialogue-v0.1/validate.mjs`.
