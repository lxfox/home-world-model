# Change Impact Revalidation v0.1 oracle

This oracle tests the [Change Impact Revalidation Profile](../../../spec/profiles/change-impact-revalidation/v0.1/README.md) with 36 semantic cases and 40 forbidden inferences. It covers change semantics and time, explicit dependency closure, identity versus assumption impact, historical retention, membership/governance/privacy changes, and bounded follow-up.

Run `node conformance/scenarios/change-impact-revalidation-v0.1/validate.mjs`.
