# 产品愿景

## 为什么存在

今天的家庭智能知识往往被锁在一次装修、一组设备、一个品牌平台或一个 Agent 中。设备替换后能力和历史断裂，平台迁移后规则与语义丢失，新的 Agent 只能重新观察和猜测这个家庭。

Home World Model 希望让家庭拥有一份独立于任何单一 Agent、设备厂商和运行平台的持久知识。授权 Agent 可以从它理解空间、绑定 IEC 的功能位置、设备、测量区域、需求、约束、观测、预期影响和历史，同时清楚知道每个判断的来源、时效、冲突与权限边界。

## 长期影响

如果这一模型成立，家庭智能将从“某个平台记住了一些自动化”转向“家庭自身拥有可迁移的数字连续性”：

- 装修前，设计 Agent 能根据空间、使用需求和设备影响模型提出可验证的电力、传感器与设备方案；
- 调试时，安装位置、功率、照度、温度、噪声、能耗等预期可以通过实际观测校准；
- 运行时，不同 Agent 能在授权范围内取得同一家庭现实的目的化视图；
- 设备、平台或 Agent 更换后，功能角色、家庭意图、证据和经验仍然存在；
- 人可以确认、反驳和修正 Agent 的认识，修订过程本身也可追溯。

HWM 的最终价值不是拥有最大的本体，而是形成一个足够小、能被不同实现共同读懂和检验的家庭—Agent 互操作共识。
