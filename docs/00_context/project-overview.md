# 项目概览

## 项目名称

**Home World Model（HWM）**。

英文为规范语言，简体中文为必需的版本匹配镜像。身份、语言和许可证决策见 [`ADR-002`](../05_decision/ADR-002-project-identity-language-and-licensing.md)。

## 项目简介

HWM 是一套面向家庭持久知识的开放、组合式互操作 Profile。它使不同的授权 Agent 能够读取同一个家庭知识包，取得受用途与时间约束的 World View，并交换可追溯的 Claim、证据和 Action Trace。

HWM 不替代 Matter、Home Assistant、建筑模型或 Agent 自身的推理系统。它关注这些系统之间尚未稳定解决的问题：如何让家庭的空间、绑定 IEC 的功能位置、设备身份、需求、观测、预期影响、权限和历史在设备与 Agent 更换后仍然连续。

## 背景

现有智能家居主要在解决连接、发现、控制和单平台自动化。进入 Agent 时代后，控制之前还需要一个可携带、可授权、可质疑的家庭事实层：Agent 必须知道“它认为的事实来自哪里、在什么时间和用途下有效、是否有冲突，以及行动后究竟发生了什么”。

三维模型是重要视图和证据入口，但不是唯一事实源；用户确认、传感观测、设备声明、设计图、调试记录和学习模型都可以形成不同认知依据的 Claim。

## 当前阶段

项目处于**核心模型讨论候选与实现验证准备阶段**。公开叙事 v0.1 已形成；Core v0.1 的概念边界已初步收敛，但尚未成为采用标准。

## 产品目标

先用两个独立实现验证一个最小 HWM 互操作 Profile，再决定哪些部分值得进入正式规范、参考工具或后续产品。第一个可信成果是可移植的家庭知识包和一致性测试，而不是完整智能家居产品。
