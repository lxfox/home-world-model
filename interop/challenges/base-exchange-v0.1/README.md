# Base Exchange independent implementation challenge

This public placeholder records the intended participation surface for an independent Base Exchange implementation challenge.

The challenge package and intake process are not published yet. Until a content-bound release, named reviewers, submission terms, and a verified discussion endpoint exist, this page must not be treated as an open call for submissions or as evidence of independent interoperability.

For now, implementers can inspect the [Base Exchange conformance set](../../../spec/conformance-sets/base-exchange/v0.1/README.md) and discuss ambiguities through this repository's GitHub Discussions.
